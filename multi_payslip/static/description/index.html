Multiple Payslip

The Multiple Payslip module helps user to create payslip for multiple employees. Whereas the default functionality only offered facility of creating payslip for each employee separately.

Also, the default Odoo payslip feature allowed payslip generation of one month. While with this module user can create payslip for multiple employees as well as can avail the facility of creating payslip of more than one month for each user. The basic highlights of this modules are:

  • Can allow user to create payslip for multiple employees at a time.
  • User can create payslip of more than one month for each employee.

Prerequisites before Installation

No prerequisites required.

Configurations

No configurations.

Under the Employee Tab you can select multiple employees whose payslip are to be generated.

After selecting multiple employees, click on the “Create Payslip” Action under “Action” button.

A “Create Payslip” wizard will popup after having been clicked on create payslip action. Mention the “Start” and “End” date for payslip.

Click on Confirm to generate payslip for selected employees. But the status for each payslip will be in draft.

In Odoo you need to confirm the payslip to bring them to the done state. For that you need to navigate to- “Employee Payslips” tab and click on “Multi Payslip Confirm” action.

As soon as you mark the selected payslips as confirm, you can see the status of selected payslips are converted to “Done” from “Draft”.
